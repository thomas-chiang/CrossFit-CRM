## Features
- Membership administration:
  - Validate users
  - Update points
  - Update roles
   
- Workout customization:
  - Manange movements
  - Manange workouts

- Performance analysis:
  - Manange performances
  - Analyze performances
  - Compete in Leaderboards

- Course management:
  - Manange Courses
  - Enroll Courses
  - Checkout Courses
